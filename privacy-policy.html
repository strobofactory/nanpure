プライバシーポリシー - ナンプレ (Number Place Puzzle)

最終更新日: 2026年2月21日

収集する情報

本拡張機能は、個人情報を一切収集しません。

データの保存

ゲームの進行状況（クリア回数、ベストタイム）は、お使いのブラウザのローカルストレージにのみ保存され、外部サーバーへ送信されることはありません。

第三者への提供

本拡張機能は、いかなるデータも第三者に提供しません。

外部サービスへの接続

本拡張機能は、Googleフォント（fonts.googleapis.com）への接続を行います。これはフォントの読み込みのみを目的としています。

お問い合わせ

ご質問がある場合は、Chrome Web Storeの拡張機能ページからお問い合わせください。

Privacy Policy - Number Place Puzzle

Last updated: February 21, 2026

Information We Collect

This extension does not collect any personal information.

Data Storage

Game progress (clear count, best times) is stored only in your browser's local storage and is never transmitted to any external server.

Third-Party Sharing

This extension does not share any data with third parties.

External Connections

This extension connects to Google Fonts (fonts.googleapis.com) solely for the purpose of loading fonts.

Contact

If you have any questions, please contact us through the extension page on the Chrome Web Store.
